genai Release:

Command Line AI Assistant via command line or shell

Github Repository


What is genai?
Genai is an efficient and lightweight artificial intelligence inference tool that can be accessed directly from the command line. This makes it an excellent choice for developers and AI enthusiasts who prefer quick and straightforward AI model interactions.

One of the strengths of Genai is its ability to utilize HuggingFace's Inference API. That API is a robust platform known for its wide array of pre-trained AI models and its capability to facilitate seamless AI inferences. Specifically, GenAI leverages the Mistral-7B-Instruct-v0.2 model hosted on HuggingFace by accessing a serverless endpoint. This means that users can execute AI inferences without worrying about managing the underlying infrastructure, leading to a hassle-free experience.



Why genai?
The serverless nature of the HuggingFace Inference API, specifically in regard to the Mistral-7B-Instruct-v0.2 model, ensures that genai will always have hosted, speedy inference. This makes GenAI a scalable and cost-effective solution for running AI inferences from extremely lightweight machines.

Genai is of particular interest for those who frequently use the command line for their work. It addresses the need for quick answers and solutions in the daily routine of developers, system administrators, and other technical professionals all from the command line. Before Genai, finding a solution that was both free of cost and light on resources was challenging. Now, with GenAI, users can obtain fast and accurate results without leaving their command line environment.



Installation:
Installation is very simple, and only requires two steps. Installing the package, and setting the API key.

You will need to generate a HuggingFace Inference API key. This can be done here.

Installing via pip

pip install genai-cmd

Setting up API key

genai -S hf_xxxxxxxxxxxxxxxxxx


Usage:
Here are some basic intructions on the usage of how to operate the genai tool. Also included in this section are the relatvent flags.


Flags:
-t, --text-data      Used to ask a question to genai directly without entering the shell.
-r, --raw-data       Same as -t without text formatting.

-c, --context        Used to set the temporary context genai answers prompt(s) under. Works with shell. (ex. "You are a pirate", "You are an angry old man")
-l, --length         Used to set the maximum length of response. Default is 100 characters. Works with shell.
-s, --seconds        Used to set the maximum time a response will take to generate. Default is 3. Works with shell.

-C, --reset-context  Used to set the default context genai answers all prompts under. (ex. "You are a pirate", "You are an angry old man")
-S, --set-key        Used to set the HuggingFace Inference API key needed for genai to operate.

Who stands to benefit?
The people who are most likely to gain benefit from Genai are individuals and professionals who frequently interact with the command line. This tool significantly enhances their capability to perform AI-driven Q&A directly from the command line interface; streamlining their workflows and increasing productivity. Here are some groups that stand to benefit significantly from GenAI:

Developers and Programmers:
These professionals often rely on quick access to tools. Genai provides them with fast AI inference, helping to speed up the development process and improve quality of life.

System Administrators and DevOps Teams:
These users can leverage genai for quick command syntax, logical system analysis, and soon file integration! This will make it easier than ever to manage and troubleshoot systems efficiently.

Data Scientists and Analysts:
With Genai, they can quickly query models, run data analytics, and get predictions or insights from data, facilitating faster decision-making and research.

Students and Hobbyists:
Those learning programming or exploring AI can use Genai to experiment with fast and free AI inference as well as gaining a level of practical experience with AI inference, enhancing their learning journey.

The feature of opening Genai shells with a set agent context is particularly powerful because it allows users to maintain a session-like experience where the context of their queries is preserved over their use of that shell.

By lowering the barrier to AI inference and making it accessible directly from the command line, Genai provides quick and easy access to powerful AI capabilities, making it a valuable tool for a wide range of users across different sectors.
